# CEO Plan: wrong-answer.ai — 학습 DNA 플랫폼

## Vision

### 10x Check
"한국 고등학생 1인당 학습 DNA를 구축하는 인텔리전스 플랫폼"

매번 시험지를 올릴 때마다 데이터가 쌓인다. 1년 후면 학생별로 개념 수준 지도, 오답 패턴, 진단→처방→검증 루프가 데이터로 존재한다. 콴다가 못 가진 것: "학생별 시간 누적 데이터." 콴다는 문제를 풀고 끝이다. wrong-answer.ai는 실수를 기억하고 추적한다.

### Platonic Ideal
고1 최서연이 모의고사를 받아 들었다. 스마트폰으로 시험지를 찍었다. 30분 후 분석 완료. 5일 후 앱이 알림: "서연아, 수열 파트 오늘 복습할 시간이에요." 탭하면 새 문제 3개. AI: "이 문제에서 어디서 막혔어?" Socratic 대화 → 복습 완료 → 스트릭 +1. 부모 폰: "서연이가 오늘 복습을 완료했습니다. 수열 +15점 예상."

## Accepted Scope

### Core
- 사진/이미지 업로드 → GPT-4o Vision 오답 분석 → 히스토리 저장 (RAG)
- 전 과목 지원: 국어·영어·수학·과학·사회 + 과목별 프롬프트 템플릿
- 오답 원인 3 카테고리 자동 분류: 계산 실수 / 개념 미숙지 / 출제 의도 파악 실패

### AI 튜터
- 기본 모드: 풀이 과정 + 핵심 개념 단계별 설명
- Socratic 모드: 답 대신 질문으로 유도 (학생별 on/off, 기본값 on)
- 유사 문제 자동 생성 (동일 난이도, 동일 개념 변형)

### Habit Loop
- SM-2 알고리즘 기반 간격 반복 복습 스케줄링 (next_review_at 컬럼)
- 일일 복습 알림: 앱 푸시 + 카카오톡
- 스트릭 시스템 (연속 학습 일수, 최장 기록, 뱃지)
- 진행도 시각화: 주간 복습 완료율, 약점 개선 추이

### 학부형 기능
- 앱 내 주간 학습 리포트 (과목별 오답률, 스트릭, 개선된 개념)
- 카카오톡 알림톡 자동 발송 (매주 월요일 오전 9시)
- 배우자 계정 연결 (학부형 2인 공유 뷰)

### 결제
- Free: 월 10문제 분석, 기본 AI 챗봇
- Pro: 무제한 분석, SM-2 스케줄링, Socratic 모드, 학부형 카카오 리포트
- TossPayments 연동 (기존 코드 활용)
